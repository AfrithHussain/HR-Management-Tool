## Color Scheme

```css
- Main: #0a0a0a (black)
- Cards: rgba(31, 41, 55, 0.5) with backdrop-blur
- Sidebar: rgba(17, 24, 39, 0.5)
- Inputs: #111827 (gray-900)

/* Borders */
- Default: #374151 (gray-700)
- Focus: #10b981 (emerald-500)

/* Text */

```
