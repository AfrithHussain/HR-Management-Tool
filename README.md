# My Personal Project

A Next.js application with Firebase authentication and Firestore database.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
Create a `.env.local` file with your Firebase and Gemini API credentials (see `.env.local` for reference)

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Features

- Firebase Authentication
- Firestore Database
- Admin Dashboard
- User Management
- Content Upload & Search
- Quiz System

## Tech Stack

- Next.js 16
- React 19
- Firebase
- Tailwind CSS
- Google Gemini AI
